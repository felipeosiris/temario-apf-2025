## Cómo Publicar en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube los archivos del sitio (`index.html`, `styles.css`, `README.md`).
3. Ve a las configuraciones del repositorio.
4. Habilita GitHub Pages:
   - Source: Escoge la rama principal.
   - Directory: Escoge `/root`.
5. Accede al enlace generado por GitHub Pages.
